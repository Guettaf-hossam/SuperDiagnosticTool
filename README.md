# Super Diagnostic Tool

## Overview
This is a comprehensive Windows diagnostic tool designed to help users diagnose and fix system issues. It includes features for monitoring CPU/RAM usage, managing services, checking disk health, and more.

## Features
- **System Monitoring**: Real-time tracking of CPU and Memory usage.
- **Service Management**: identify and restart failed services.
- **Disk Health**: Scan and report on disk status.
- **Superman Eye**: Advanced diagnostic mode for deep system analysis.
- **Optimization**: Tools to clean up disk space and optimize system performance.

## Usage
Run the script `super_diagnose_v2.py` to start the diagnostic interface.

```bash
python super_diagnose_v2.py
```
